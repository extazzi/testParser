Задание:
Часть 1.
Написать 2 класса.
1) Подключается по указанному адресу и парсит сайт. Он должен получить все уравнения со всех страниц. Желательно показать умение работать с регулярными выражениями.
2) Класс является оберткой для таблицы в базе данных. Должен содержать методы для записи и чтения из базы данных. Конструктор класса принимает параметры подключения к БД. Так же конструктор выполняет запрос к БД, который создает таблицу для своей работы, если такой еще нет.

Результат выполнения:
Объект первого класса собирает все уравнения с сайта и использую объект второго класса записывает их в базу данных.

Часть 2.
Реализовать 2 класса.
1) Первый класс содержит метод, который принимает строку, математическое выражение с использованием скобок и основных действий (- + * / ). На выходе выдает строку, выражение переведенное в обратную польскую запись. Необходимо учитывать, что входящее выражение может иметь пробелы между элементами выражения, а может и не иметь. Так же возможны ошибки при построении уравненияс (пропущенный операнд, скобка или деление на ноль).
        Пример:
        2 + 8 * 4        ==> 8 4 * 2 +
        (2 + 8) * 4      ==> 2 8 + 4 *
        (8 + 2           ==> ошибка, нет закрывающей скобки
        8 +              ==> ошибка, нет второго операнда
2) Второй класс содержит метод, который принимает строку в формате обратной польской записи и решает ее.
        Пример:
        8 4 * 2 +        ==> 34
        2 8 + 4 *        ==> 40

Результат выполнения:
После первой части в базе лежат уравнения. Используя класс из первой части достать выражения из БД, оброботать и записать результат в БД.

В качестве СУБД используется MySQL, конфигурация находится в _confiq.php. Работа осуществляется с пом. PDO.

Использовал:
PHP 5.6;
MySQL - 5.5.
